# Whisper Wall

![Whisper Wall Screenshot](https://i.imgur.com/JQ8XZ1l.png)

Whisper Wall is an anonymous confession platform where users can share their thoughts without revealing their identity. Built with Firebase and featuring real-time updates, AI enhancements, and interactive elements.

## Features

✨ **Real-time Updates**  
- See new posts instantly without refreshing
- Live typing indicators show when others are composing
- Online user counter

🤖 **AI Enhancements**  
- Transform your posts with AI:
  - Make it **Spicy** 🌶️
  - Make it **Funny** 😂
  - Make it **Mysterious** 🕵️‍♂️
  - Make it **Dramatic** 🎭

🎭 **Anonymous Posting**  
- No accounts needed
- Complete privacy protection
- All posts are untraceable

💖 **Interactive Elements**  
- React to posts with emojis (❤️, 😂, 😮, etc.)
- Smooth animations for new posts
- Dark/light mode toggle

## Technologies Used

- **Frontend**: 
  - HTML5, CSS3, JavaScript
  - [Quill.js](https://quilljs.com/) rich text editor
- **Backend**: 
  - [Firebase](https://firebase.google.com/):
    - Firestore (database)
    - Realtime Database (presence system)
    - Authentication (anonymous sign-in)
- **AI**: 
  - [Google Vertex AI](https://cloud.google.com/vertex-ai) (text-bison model)
